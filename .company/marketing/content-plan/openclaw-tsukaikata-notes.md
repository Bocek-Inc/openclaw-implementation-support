# 使い方記事 実演メモ

## レシピ①: AI最新情報の毎朝自動配信

### 設定内容
- ジョブ名: AI・テックニュース朝刊
- スケジュール: 毎朝 8:10 JST
- 配信先: #openclaw-news（Slack）
- モデル: Claude Sonnet 4.6
- 収集カテゴリ: 生成AI / AIエージェント / 業務効率化ツール
- 最大7件、1〜2行要約＋URL、日本語
- Brave Search API未設定。The Verge・TechCrunch等を直接スクレイピングで収集

### しゅーへいさんの所感（原文）
- 便利だと思った
- ネットを検索しなくても自分が欲しい情報だけがまとめて手に入る
- ついでにネットサーフィンしちゃって時間を浪費するリスクが抑えられる
- 情報の質や量もちょうどよかった
- Web APIを使ったらもっと幅広く情報が手に入るかも
- こっちから聞かなくても決まった時間に送ってくれるのがありがたい
- 聞くのがひと手間だったので（ChatGPTとの違い）

### 記事で使えるポイント
- 「ネットサーフィンによる時間浪費の防止」→ 共感を呼ぶフレーズ
- 「聞かなくても届く」→ ChatGPTとの明確な違い
- 設定は実質5分程度（プロンプトを送るだけ）
