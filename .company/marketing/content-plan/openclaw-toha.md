### h2: OpenClawのセットアップ手順【実践】

#### h3: VPSとは？（簡潔な補足）
- 2〜3行で「インターネット上に借りる自分専用のPC」と説明
- 月額数百円〜、24時間稼働、OpenClawに最適であることを伝える
- ※非エンジニア読者の離脱防止のために必要

#### h3: 動作環境の3つの選択肢
| 選択肢 | メリット | デメリット |
|--------|---------|-----------|
| 自分のPC | 無料、すぐ始められる | PC OFF で停止、セキュリティリスク |
| Mac mini等 | 常時稼働、隔離可能 | 初期投資500ドル〜、品薄 |
| VPS（推奨） | 月数ドル〜、24/7、問題が隔離 | 月額コスト |
- **素材**: 動画1-セクション2、動画3-セクション5
- **🔧 しゅーへいさん作業**: 実際にセットアップした画面のスクショがあるとオリジナルコンテンツに

#### h3: VPS（Hostinger）でのセットアップ手順
- OpenClaw専用ページ（hostinger.com/vps/docker/openclaw）からワンクリックデプロイ
- プラン選択：KVM 2推奨（CPU2 / RAM 8GB / 100GB SSD）
  - まず試したい方は1ヶ月プランがおすすめ（最小コストで始められる）
  - 長期利用が決まっている方は24ヶ月が月額最安
- 「Ready to use AI（すぐ使えるAI）」はオフにする → 後で自分のAPIキーを設定した方がコスパ良い
- 自動バックアップはオン推奨（月3ドル程度、復元が楽）
- ゲートウェイトークンの取り扱い
- LLM APIキー設定（Anthropic推奨：40ドルチャージでTier 2）
- 支出ガードレール設定
- **素材**: 動画1-セクション3,4
- **🔧 スクショ（撮影済み）**:
  - OpenClaw専用ページのトップ画面
  - KVM 2を赤枠で囲んだプラン選択画面

### h2: OpenClawのセキュリティリスク

#### h3: なぜセキュリティ対策が必須なのか
- ターミナルコマンド・ファイル読み書き・Web閲覧・メッセージ送信が可能
- 理解のない人は「使わない方がよい」（安野高弘氏の警告）
- **素材**: 動画1-セクション6、動画3-セクション9

#### h3: 主なリスク3つ
1. **情報漏洩・ファイル改ざん**: 気づかないうちに外部送信、ファイル削除・変更
2. **脆弱性の悪用**: リモートから悪意ある操作を実行させられるリスク
3. **ClawHub（スキルマーケット）のリスク**: 約7%のスキルに脆弱性、人気スキルでも問題あり
- **素材**: 動画3-セクション9

#### h3: 安全に使うための対策
- セキュリティドキュメントをボット自身に実装させる
- 最小権限の原則（メッセージ送信・ファイル削除は事前承認制）
- ガードレール設定（3回失敗で停止、実行時間上限10分）
- 小さく始める（Telegramだけ、スキル1〜2個から）
- 大切なファイル・機密情報のあるマシンでは使わない
- **素材**: 動画1-セクション6、動画3-セクション9

#### h3: OpenClaw本体は無料（オープンソース）
- ソフトウェア自体は完全無料
- お金がかかるのは「実行環境」と「LLM API」の2つ

#### h3: 実行環境のコスト
| 環境 | 月額目安 |
|------|---------|
| 自分のPC | 0円（電気代のみ） |
| VPS（小規模） | 月5〜12ドル |
| Mac mini | 初期投資500ドル〜 |

#### h3: LLM APIのコスト
| 利用スタイル | 月額目安 |
|------------|---------|
| 低価格LLM（Haiku等） | 月5〜20ドル |
| 標準LLM（Sonnet等） | 月30〜80ドル |
| フラッグシップ（Opus等） | 月100〜300ドル以上 |
| ローカルLLM | 0円（マシンスペック必要） |

- モデルルーティングでコスト最適化が可能
